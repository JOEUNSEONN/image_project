🔮 Rich 타로 운세 콘솔 프로그램

Rich 라이브러리를 활용해 만든 감성 콘솔 기반 타로 카드 뽑기 프로그램입니다. 실행하면 오늘의 운세를 무작위로 뽑아, 마치 타로 카드처럼 꾸며서 보여줍니다.

✨ 주요 기능

✅ Rich로 감성적인 콘솔 카드 출력
✅ 하루 운세 무작위 추천
✅ 이모지와 색상으로 시각적 효과 제공
✅ 입력형 맞춤 운세 기능 추가

💻 코드 예시


from rich import print
from rich.panel import Panel
from rich.align import Align
import random

fortunes = ["휴식이 필요해요 😴", "좋은 일이 생깁니다 🎁"]
selected = random.choice(fortunes)
print(Panel(Align.center(selected), title="오늘의 운세"))

🖼 실행 결과 예시

🔮 당신의 타로 메시지
😴 휴식이 필요한 하루입니다. 너무 무리하지 마세요.

🧙 입력 기반 운세 기능 추가

사용자가 성별, 양력/음력, 생년월일을 직접 입력하면
맞춤형 운세 메시지를 출력해주는 콘솔 프로그램입니다.

📌 실행 예시


성별을 입력하세요 (m/f): f
양력 또는 음력을 입력하세요 (solar/lunar): solar
생일을 입력하세요 (예: 2001-06-11): 2001-06-11

🔮 오늘의 운세
🎁 깜짝 선물 같은 일이 생길 수 있어요!

⚙️ 실행 방법

  1. Python 설치
  2. Rich 설치: pip install rich
  3. 타로 프로그램 실행: python tarot_fortune.py
  4. 입력형 프로그램 실행: python fortune_console.py

📂 프로젝트 파일 구성

  • tarot_fortune.py – 번호 선택 기반 콘솔 프로그램
  • fortune_console.py – 입력 기반 맞춤 운세 콘솔 프로그램
  • index.html – 소개용 웹사이트
  • style.css – 사이트 스타일시트
  • images/ – 결과 예시 이미지 폴더
